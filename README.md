# milu - (milu-backend)

## General

milu - an app to use todos

### Requirements

* java 21
* h2 or postgres database

### Install

```shell
./mvnw clean install
```

### Run

```shell
java -jar target/milu-backend-0.1.jar server src/main/resources/example-server-h2.yml
```
